# AGIT Documentation Website

Official documentation site for [AGIT](https://github.com/agit-stuff/agit), the AI-native Git wrapper.

**Live site: https://agit-stuff.github.io/**

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
